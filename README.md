# AiChatbot
A full-stack AI chatbot platform built using the MERN stack, OpenRouter API, JWT authentication, MongoDB, and Razorpay integration.

## Live Demo
https://your-vercel-link.vercel.app

## Tech Stack

Frontend:
- React
- Vite
- Tailwind CSS

Backend:
- Node.js
- Express.js

Database:
- MongoDB

Authentication:
- JWT

AI:
- OpenRouter API

Payments:
- Razorpay
